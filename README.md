# 🚀 Next — Простой мессенджер

Next — это полнофункциональное веб-приложение для обмена сообщениями, созданное для общения между пользователями в режиме реального времени.

Этот проект был разработан как портфолио для демонстрации интеграции бэкэнда и фронтенда, аутентификации и функциональности чата.

# ✨ Особенности
 🔐 Регистрация и аутентификация пользователя 
 
 👤 Профили пользователей
 
 💬 Личные чаты один на один
 
 📩 Отправка и получение сообщений
 
 🕓 История сообщений хранится в базе данных
 
 🔄 Автоматическое обновление чата
